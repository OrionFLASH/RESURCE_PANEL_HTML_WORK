Примеры кнопок с типами

PROM - Зелено-салатовые оттенки
Доступна
Пример кнопки PROM (доступна) - яркий зеленый градиент
Яркий зеленый градиент слева с эффектом свечения
Недоступна
Пример кнопки PROM (недоступна) - тусклый зеленый
Тусклый светло-зеленый градиент с пониженной непрозрачностью
PSI - Желто-сиреневые оттенки
Доступна
Пример кнопки PSI (доступна) - яркий желтый градиент
Яркий желтый градиент слева с эффектом свечения
Недоступна
Пример кнопки PSI (недоступна) - тусклый желтый
Тусклый светло-желтый градиент с пониженной непрозрачностью
IFT - Бледно-голубые оттенки
Доступна
Пример кнопки IFT (доступна) - яркий голубой градиент
Яркий голубой градиент слева с эффектом свечения
Недоступна
Пример кнопки IFT (недоступна) - тусклый голубой
Тусклый светло-голубой градиент с пониженной непрозрачностью
OTHER - Без дополнительной раскраски
Доступна
Пример кнопки OTHER (доступна) - стандартное отображение
Стандартное отображение без цветовой индикации слева
Недоступна
Пример кнопки OTHER (недоступна) - стандартное отображение
Стандартное отображение без цветовой индикации слева
Сравнение всех типов (доступные)
PROM - Зеленый
PSI - Желтый
IFT - Голубой
OTHER - Без индикации
Сравнение всех типов (недоступные)
PROM - Зеленый (тусклый)
PSI - Желтый (тусклый)
IFT - Голубой (тусклый)
OTHER - Без индикации
