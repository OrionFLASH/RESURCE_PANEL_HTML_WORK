## Критические проблемы (требуют немедленного исправления)

### 1. Дублирование функции `generateNewsScript`
**Местоположение:** Строки 1493-1541 и 1583-1631

**Проблема:** Функция `generateNewsScript` определена дважды в коде, что может привести к конфликтам и неопределенному поведению.

**Решение:** Удалить одну из версий функции, оставить только одну реализацию.

### 2. Дублирование ID `psi2-alpha-btn`
**Местоположение:** Строки 461 и 465

**Проблема:** Одинаковый ID используется для двух разных элементов, что нарушает стандарты HTML (ID должны быть уникальными).

**Решение:** 
- Изменить ID одного из элементов (например, `psi2-alpha-btn-2`)
- Или использовать классы вместо ID для группировки элементов

---

### 3. Дублирование CSS-свойства `overflow-y`
**Местоположение:** Строки 72 и 74

**Проблема:** Свойство `overflow-y: auto` указано дважды для класса `.groups-viewport`.

**Решение:** Удалить одно из дублирующихся свойств.

```css
.groups-viewport {
    flex: 1 1 auto;
    min-height: 0;
    min-width: 0;
    height: 100%;
    overflow-y: auto; /* Оставить только одно */
    max-height: calc(100vh - 160px);
}
```

### 4. Устаревший API копирования в буфер обмена
**Местоположение:** Строки 1222, 1404, 1486, 1580, 1793

**Проблема:** Используется устаревший метод `document.execCommand('copy')`, который помечен как deprecated.

**Решение:** Заменить на современный API `navigator.clipboard.writeText()`.

**Пример замены:**
```javascript
// Старый код:
sgResult.select();
document.execCommand('copy');

// Новый код:
navigator.clipboard.writeText(sgResult.value).then(() => {
    sgCopyBtn.textContent = 'Скопировано!';
    setTimeout(() => sgCopyBtn.textContent = 'Скопировать', 1200);
}).catch(err => {
    console.error('Ошибка копирования:', err);
});
```

### 9. Отсутствие обработки ошибок
**Проблема:** Многие fetch-запросы не имеют обработки ошибок и таймаутов.

**Решение:** Добавить обработку ошибок и таймауты:

```javascript
async function fetchWithTimeout(url, options = {}, timeout = 5000) {
    const controller = new AbortController();
    const id = setTimeout(() => controller.abort(), timeout);
    
    try {
        const response = await fetch(url, {
            ...options,
            signal: controller.signal
        });
        clearTimeout(id);
        return response;
    } catch (error) {
        clearTimeout(id);
        if (error.name === 'AbortError') {
            throw new Error('Запрос превысил время ожидания');
        }
        throw error;
    }
}
```

### 11. Навигация с клавиатуры
**Проблема:** Модальные окна не закрываются по нажатию Escape и не ловят фокус.

**Решение:** Добавить обработку клавиатуры:

```javascript
// Закрытие модального окна по Escape
document.addEventListener('keydown', function(e) {
    if (e.key === 'Escape') {
        const openModal = document.querySelector('.modal.show');
        if (openModal) {
            openModal.classList.remove('show');
        }
    }
});

// Ловушка фокуса внутри модального окна
function trapFocus(modal) {
    const focusableElements = modal.querySelectorAll(
        'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
    );
    const firstElement = focusableElements[0];
    const lastElement = focusableElements[focusableElements.length - 1];
    
    modal.addEventListener('keydown', function(e) {
        if (e.key === 'Tab') {
            if (e.shiftKey) {
                if (document.activeElement === firstElement) {
                    lastElement.focus();
                    e.preventDefault();
                }
            } else {
                if (document.activeElement === lastElement) {
                    firstElement.focus();
                    e.preventDefault();
                }
            }
        }
    });
}
```

### 13. Отсутствие debounce/throttle
**Проблема:** Проверка ссылок каждые 2 минуты может быть избыточной, нет ограничения на количество одновременных запросов.

**Решение:** Добавить debounce/throttle и ограничение параллельных запросов:

```javascript
// Throttle для проверки ссылок
function throttle(func, limit) {
    let inThrottle;
    return function() {
        const args = arguments;
        const context = this;
        if (!inThrottle) {
            func.apply(context, args);
            inThrottle = true;
            setTimeout(() => inThrottle = false, limit);
        }
    };
}

// Ограничение параллельных запросов
async function checkCardsWithLimit(cards, limit = 5) {
    const results = [];
    for (let i = 0; i < cards.length; i += limit) {
        const batch = cards.slice(i, i + limit);
        const batchResults = await Promise.allSettled(
            batch.map(card => checkCard(card))
        );
        results.push(...batchResults);
    }
    return results;
}
```

```javascript
function validateTournamentIds(input) {
    const ids = input.split(/[^a-zA-Z0-9\-_]+/g)
        .map(s => s.trim())
        .filter(Boolean);
    
    if (ids.length === 0) {
        throw new Error('Не указано ни одного Tournament ID');
    }
    
    // Проверка формата
    const invalidIds = ids.filter(id => !/^t_\d{2}_\d{4}-\d+_\d{2}-\d+_\d+_\d+$/.test(id));
    if (invalidIds.length > 0) {
        throw new Error(`Некорректные ID: ${invalidIds.join(', ')}`);
    }
    
    return ids;
}
```

### 22. Логирование
**Проблема:** Отсутствие структурированного логирования для отладки.

**Решение:** Создать систему логирования:

```javascript
const Logger = {
    levels: {
        DEBUG: 0,
        INFO: 1,
        WARN: 2,
        ERROR: 3
    },
    
    log(level, message, data = null) {
        const timestamp = new Date().toISOString();
        const levelName = Object.keys(this.levels).find(
            key => this.levels[key] === level
        );
        
        const logMessage = `[${timestamp}] [${levelName}] ${message}`;
        
        if (data) {
            console[levelName.toLowerCase()](logMessage, data);
        } else {
            console[levelName.toLowerCase()](logMessage);
        }
    },
    
    debug(message, data) { this.log(this.levels.DEBUG, message, data); },
    info(message, data) { this.log(this.levels.INFO, message, data); },
    warn(message, data) { this.log(this.levels.WARN, message, data); },
    error(message, data) { this.log(this.levels.ERROR, message, data); }
};
```

### 24. Оптимизация модальных окон
**Проблема:** Дублирование кода для закрытия модальных окон.

**Решение:** Создать универсальную систему управления модальными окнами:

```javascript
const ModalManager = {
    modals: new Map(),
    
    register(id, options = {}) {
        const modal = document.getElementById(id);
        if (!modal) return;
        
        const config = {
            closeOnBackdrop: true,
            closeOnEscape: true,
            trapFocus: true,
            ...options
        };
        
        this.modals.set(id, { modal, config });
        this.setupEventListeners(id);
    },
    
    setupEventListeners(id) {
        const { modal, config } = this.modals.get(id);
        
        // Закрытие по клику вне окна
        if (config.closeOnBackdrop) {
            modal.addEventListener('click', (e) => {
                if (e.target === modal) {
                    this.close(id);
                }
            });
        }
        
        // Закрытие по Escape
        if (config.closeOnEscape) {
            document.addEventListener('keydown', (e) => {
                if (e.key === 'Escape' && modal.classList.contains('show')) {
                    this.close(id);
                }
            });
        }
    },
    
    open(id) {
        const { modal, config } = this.modals.get(id);
        modal.classList.add('show');
        if (config.trapFocus) {
            this.trapFocus(modal);
        }
    },
    
    close(id) {
        const { modal } = this.modals.get(id);
        modal.classList.remove('show');
    }
};
```

### 25. Кэширование результатов проверки
**Проблема:** Отсутствие кэширования результатов проверки доступности ссылок.

**Решение:** Добавить кэширование:

```javascript
const LinkChecker = {
    cache: new Map(),
    cacheTimeout: 5 * 60 * 1000, // 5 минут
    
    async check(url) {
        // Проверка кэша
        const cached = this.cache.get(url);
        if (cached && Date.now() - cached.timestamp < this.cacheTimeout) {
            return cached.available;
        }
        
        // Проверка доступности
        const available = await this.performCheck(url);
        
        // Сохранение в кэш
        this.cache.set(url, {
            available,
            timestamp: Date.now()
        });
        
        return available;
    },
    
    async performCheck(url) {
        // Логика проверки
        try {
            const response = await fetch(url, { 
                method: 'HEAD', 
                mode: 'no-cors',
                signal: AbortSignal.timeout(5000)
            });
            return true;
        } catch {
            return false;
        }
    },
    
    clearCache() {
        this.cache.clear();
    }
};
```
